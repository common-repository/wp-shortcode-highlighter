== Description ==

This plugin highlights your shortcodes in the TinyMCE editor, making them easier to see.


== Installation ==

1. Install the plugin by uploading the zip file (Plugins > Add New > Upload)
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to the Settings > Shortcode Highlighter
